BitCrusher with AudioParam

A BitCrusher example from the Web Audio API specification, but modified to demonstrate AudioParam automations. The sound source is a sawtooth oscillator at 5000Hz. The demo runs for 8 seconds.

See Chrome Developers Article: Enter Audio Worklet for more information.
